LINGO ONLINE V3 CLEAN

STARTEN
1. Open in deze map een opdrachtprompt.
2. Typ: npm install
3. Typ daarna: npm start
4. Open in Chrome: http://localhost:3000

TESTEN MET 2 APPARATEN
- Maak op apparaat 1 een kamer.
- Tik op beide apparaten op GELUID AAN.
- Vul op apparaat 2 de kamercode in en join.
- Start de ronde op de host.

BELANGRIJK
- Geluid op mobiel werkt pas nadat je op dat apparaat op GELUID AAN hebt getikt.
- Deze versie heeft een duidelijke tik per seconde.
- Woordlengtes: 4, 5 en 6.


V3 Definitief: confetti bij winst, mobiel invoervak altijd zichtbaar, duidelijkere Oppo-geluiden.


Let op: gebruik op telefoon bij voorkeur een incognitotab of voeg ?v=pausefix2 toe als je oude cache ziet.
